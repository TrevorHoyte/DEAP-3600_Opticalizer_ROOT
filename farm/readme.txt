For performing a farm job create a new farm according to the compute canada website : https://docs.alliancecan.ca/wiki/META-Farm

1. Use bash_farm_submitter.sh to create the table.dat file in your farm directory.
2.read the META-farm documentation to setup the other farm files, additonally I have kept my verisons of the job_script.sh file and the single case.sh file as reference.
3. submit the farm run
4. add the ntp file sup using superhadd.sh
